**תיאור הפרויקט:**
פרויקט זה מציג תכנון, פיתוח וניהול של בסיס נתונים רלוציוני (PostgreSQL) עבור מערכת מסחר אלקטרוני מקצה לקצה. המערכת מנהלת לקוחות, הזמנות, מלאי מחסנים, ספקים, כוח אדם לוגיסטי וחברות שילוח, תוך הקפדה על שלמות נתונים ואילוצים עסקיים.
